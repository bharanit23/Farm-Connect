# Farm Connect
AI-powered WhatsApp platform connecting farmers, 
agricultural labourers, and equipment providers.

## Problem
Farmers in rural India lose crops every season due to 
labour shortages and no equipment access.

## Solution
WhatsApp-based platform with AI matching, mutual 
accountability, and equipment rental.

## Tech Stack
- FastAPI, Twilio, Supabase, PostgreSQL + PostGIS
